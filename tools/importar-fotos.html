Importar Productos desde Fotos

Formato del nombre del archivo:
NombreProducto_Marca_NombreTienda.jpg (mínimo) o NombreProducto_Marca_Categoria_Subcategoria_NombreTienda.jpg

Si no incluyes categoría y subcategoría, el sistema las detecta automáticamente desde el nombre del producto usando palabras clave.
Ejemplo: Colageno Hidrolizado_Natural Systems_Colageno Hidrolizado.jpg → detecta categoría "Belleza" y subcategoría "Colágeno"

Si el producto ya existe (mismo nombre + nombre tienda) se salta automáticamente.
